# Pokedex program
This python program allows you to view, change, and remove Pokemon characters in a python dictionary, sourced from an external API.

## Features
- View certain characteristics of any pokemon of your choice
- Add pokemon characters to your 'pokedex' dictionary
- Remove pokemon characters from your 'pokedex' dictionary

## Requirements
- To run this program, you need to install the following dependencies:
    - `requests` to make HTTP requests to the pokedex API
