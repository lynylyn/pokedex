# Functional Requirements
### Data Retrieval
- The user should initially be able to view the 'selection' interface - more specifically, options to edit, view, or exit the pokedex.
- The user should be able to retrieve their personal pokedex dictionary.
### User Interface
- The user must be able to access the python terminal, and have installed the 'requests' dependency.
- They must have options for user input at each stage.
